# BreakMyVM
This repository contains mini scripts made to 'break' your Virtual Machine.  
These scripts are designed for people who want hands on experience with debugging and troubleshooting systems.  



## Setup
1. Create VM
2. Take snapshot
3. Run a breaker script
4. Attempt to troubleshoot
5. Revert back to snapshot if needed

## Download
```
git clone https://github.com/Isaac-Hawks/BreakMyVM
cd BreakMyVM
```

## Important!
Do ***NOT*** run these on your host machine.  
Users are advised to take a snapshot of their VM prior to the state of running one of these scripts.
